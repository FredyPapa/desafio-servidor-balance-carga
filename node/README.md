# Desafío: Nuestra primera Base de Datos

## Instalación del proyecto

Ejecutar el código ```npm install``` para reconstruir los módulos de Node.
Crear el archivo ```.env``` con el siguiente contenido
~~~
PORT=8080
NOD_ENV=development
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=ecommerce
CORS=*
~~~
